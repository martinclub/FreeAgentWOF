Architecture & Components

A simple, event-driven Discord bot

System Flow

1. Discord Bot (discord.js)

Main listener for Button Interactions and Modal Submissions.

⬇️

2. State Manager

A simple JSON-based storage to track the current Free Agents list.

⬇️

3. UI Updater

Automatically edits the original message whenever the list changes, keeping the view in sync.

Interactions

Button: "Looking for Team"

Adds the user to the list and announces: "@Player is now a Free Agent! 🚀"

Button: "I found a team"

Opens a Modal (Pop-up) to ask for the team name. Removes the user from the list and announces: "@Player found a team! They joined [Team Name]! 🏆"
